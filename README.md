# Softskills

Ce projet est une base simple pour explorer ou développer des compétences en soft skills à travers une interface web.

## Site en ligne

[Accéder au site](https://softskills.maelg.com)

## Structure du projet

- `index.html` : Page principale de l'application web.
- `style.css` : Feuille de style pour la mise en forme de la page.
- `script.js` : Script JavaScript pour la logique côté client.

## Utilisation

Ouvrez simplement le fichier `index.html` dans votre navigateur pour démarrer l'application.
